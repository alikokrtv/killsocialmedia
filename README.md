## 📖 Hakkında

Sosyal medyada geçirdiğiniz zamanı bilinçli olarak yönetmenizi sağlayan açık kaynak tarayıcı aracı. Site engellemesi, zaman sınırı ve kullanım istatistikleri içerir.

---

## ✨ Özellikler

| | Özellik | Açıklama |
|---|---|---|
| 🚫 | **Site Engelleme** | Belirlediğiniz sosyal medya sitelerini otomatik engeller |
| ⏱️ | **Zaman Sınırı** | Günlük kullanım süresi limiti belirleyin |
| 📊 | **Kullanım İstatistikleri** | Hangi sitede ne kadar vakit harcadığınızı görün |
| ⚪ | **Whitelist** | İzin verilen siteleri özelleştirin |

---

## 🛠️ Teknolojiler

`CSS · JavaScript · Browser Extension API`

---

## 🚀 Kurulum

1. Repoyu klonlayın
2. Tarayıcıda `chrome://extensions` açın
3. "Developer mode" aktif edin
4. "Load unpacked" ile klasörü seçin
